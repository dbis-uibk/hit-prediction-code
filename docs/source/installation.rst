============
Installation
============


Stable release
--------------

To install Hit Prediction Code, run this command in your terminal:

.. code-block:: console

    $ pip install hit_prediction_code

This is the preferred method to install Hit Prediction Code, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Hit Prediction Code can be downloaded from the `Git repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://git.uibk.ac.at/dbis/hit_prediction_code


.. _Git repo: https://git.uibk.ac.at/dbis
